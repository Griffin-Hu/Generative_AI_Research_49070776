# Media Files

This folder contains media files related to the research project.

## File Organization
- Organized by media type
- Format: Description_YYYYMMDD.extension

## Media Types
- Images (.jpg, .png)
- Audio recordings (.mp3, .wav)
- Video files (.mp4)
- Presentation slides (.pptx)

## Usage Notes
- All media files have appropriate permissions for research use
- High-resolution originals are stored with compressed versions for sharing
